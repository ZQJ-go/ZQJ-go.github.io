Unity入门——界面基础

Aholic~茜2025-08-252025-09-07

Scene和Hierarchy窗口

窗口布局

右上角Layout选项可以帮助我们快速布局，这边推荐新手使用2by3布局模式，可以同时看到scene和game窗口，后期想要自己布局的话，拖动完成布局之后，点击Save Layout就可以了。

Scene（场景窗口）

场景窗口和层级窗口是息息相关的，层级窗口中看到的内容就是场景窗口中显示的对象。
渲染模式；2D、3D视图切换；光源、音效、特效开启；辅助线显示；辅助功能；搜索；场景轴向

操作物体工具栏快捷键：

  • Q：平移
  • W：移动
  • E：旋转
  • R：缩放
  • T：2D
  • Y：综合

鼠标快捷键讲解：

  • 鼠标左键：
    • 单击：选中单个物体
    • 框选：选中多个物体
    • Ctrl + 单击：多选物体
    • 长按ALT + 鼠标左键 + 移动鼠标：相对观察口中心旋转
  • 鼠标右键：
    • 移动鼠标：旋转视口
    • 按下 + WASD：漫游场景
    • 按下 + WASD + Shift：加速漫游场景
    • 长按ALT + 鼠标右键 + 移动鼠标：相对屏幕中心拉远拉近
  • 鼠标中键：
    • 滚动：对屏幕中心拉远拉近
    • 中键按下 + 移动鼠标：平移观察视口
    • 长按ALT + 滚动中键：朝哪就往哪拉近拉远

Hierarchy（层级窗口）

我们可以在Hierarchy窗口中创建或者拖动各种游戏对象，比如：模型、光源，图片，UI等；层级窗口中显示的就是一个场景中的所有对象。

在Hierarchy窗口右键或者点击左上角的+号，可以创建相应的模型对象。

快捷键：

  • F2：对象改名
  • Ctrl+C：复制
  • Ctrl+V：粘贴
  • Ctrl+D：克隆
  • Delete：删除

Game和Project窗口

Game游戏窗口：

游戏窗口，玩家能看到的游戏画面内容。
Game窗口中显示的内容是场景中的摄像机拍摄到的内容

Project工程窗口：

工程资源窗口，所有的工程资源都会在这个窗口中显示，显示的内筒为Assets文件夹中的所有内容

Inspector和Console窗口

Inspector检查窗口

用来查看场景中游戏对象冠梁的c#脚本信息
不选择场景中的游戏对象或不进行任何相关设置，该界面不会显示任何信息；当选择场景中的任意游戏对象时该界面会显示和该游戏对象关联的c#脚本信息

Console控制台窗口

用来查看调试信息的窗口，报错、警告、打印都可以显示在其中
该窗口默认不显示，可在Window——>General中开启，或者使用快捷键Ctrl + Shift + C

工具栏和父子关系

工具栏

  • 1、文件操作：新建工程，新建场景，工程打包等等；
  • 2、编辑操作：对象编辑操作相关，工程设置，引擎设置相关
  • 3、资源操作：基本等同于Project窗口中右键相关功能
  • 4、对象操作：基本等同于Hierarchy窗口中的右键相关功能
  • 5、脚本操作：Unity自带的脚本，可以添加各种系统中的脚本
  • 8、窗口：可以打开Unity各种核心系统的窗口
  • 9、帮助：检查更新，查看版本等等功能

父子关系

树形结构：子对象会随着父对象变化而变化，子对象Inspector窗口中的Transform信息是相对父对象的。
同时可以自定义改变参考系，对于子对象来说参照物可以改变为父对象或者是总的参考系。
